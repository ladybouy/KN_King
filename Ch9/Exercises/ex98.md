### Exercise 9.8

Which of the following would be valid prototypes for a function that returns nothing and has one double parameter
```
(a) void f(double x);
(b) void f(double);
(c) void f(x);
(d) f(double x);
```

Solution:
```
(a) valid
(b) valid
(c) invalid, needs type for parameter
(d) invalid, needs return type void
```
