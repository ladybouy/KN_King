### Exercise 12.11

Modify the `find_largest` function so that it uses pointer arithmetic - not subscripting - to visit array elements.

Solution:
```
int find_largest(int a[], int n)
{
    int *p, max;

    max = *a;
    for (p = a + 1; p < a + n; p++)
        if (*p > max)
            max = *p;
    return max;
}
```
