### Exercise 12.18

Write the `evaluate_position` function described in Exercise 13 of Chapter 9.  Use pointer arithmetic - not subscripting - to visit array elements. Use a single loop instead of nested loops.

Solution:
```
int evaluate_position(char board[8][8])
{
    int white = 0;
    int black = 0;
    int difference = 0;
    int *p;

    for (p = board; p < 8; p++) {
            switch (*p) {
                case 'Q': 
                    white += 9;
                    break;
                case 'R':
                    white += 5;
                    break;
                case 'B':
                    white += 3;
                    break;
                case 'N':
                    white += 3;
                    break;
                case 'P':
                    white += 1;
                    break;
                default:
                    break;
            }
        }
    }

    for (p = board; p < 8; p++) {
            switch (*p) {
                case 'q': 
                    black += 9;
                    break;
                case 'r':
                    black += 5;
                    break;
                case 'b':
                    black += 3;
                    break;
                case 'n':
                    black += 3;
                    break;
                case 'p':
                    black += 1;
                    break;
                default:
                    break;
            }
        }
    }
    difference = white - black;

    return difference;
}
```
