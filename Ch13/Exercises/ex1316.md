## Exercise 13.16

Use the techniques of 13.6 to condense the `count_spaces` function of section 13.4. In particular, replace the `for` statement by a `while` loop.

### Solution 
```
int count_spaces(const char s[])
{
    char *ptr = s;
    
    while(*ptr)
        if (*ptr == ' ')
            count++;
    return count;
}
```
