## Exercise 13.11
The Q&A section at the end of this chapter shows how the `stcmp` function might be written using array subscripting.  Modify the function to use pointer arithmetic instead.

### Solution
```
int strcmp(char *s, char *t)
{
    for (*s == *t) {
        if (*s == '\0') {
            return 0;
        }
    }
    return *s - *t;
}
```
